You can install the PHP SDK using composer with the command listed below:

#### Using Composer

```bash
composer require paymentsds/mpesa
```

#### Manual Installation

```bash
git clone https://github.com/paymentsds/mpesa-php-sdk mpesa-php-sdk
cd mpesa-php-sdk
composer install
```
